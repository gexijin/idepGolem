Interactive & reproducible analysis of gene expression data.

  • Exploratory analysis, differential expression, and pathway analyses of bulk RNA-seq or other gene-level summary data.
  • Rich, interactive visualization: clustering, KEGG pathways, genome view, etc.
  • Download HTML reports for some steps, documenting parameters and results,
  • Download dozens of publication-ready figures in minutes,
  • Download R code and data objects, which can be used in RStudio to reproduce the analysis,
  • Install locally as an R package or as a Docker container (Windows, MacOS)

For-profit organizations: contact us for licensing, local installation, or customization services.

Old versions are still usable (v2.01, v0.96).

Email Jenny (gelabinfo@gmail) or report issues on our GitHub page. iDEP is still under active development and testing. We welcome any suggestions or questions. Supported by NHGRI (National Human Genome Research Institute), NIH.
