Interactive & reproducible analysis of gene expression data:

  • Exploratory analysis, differential expression, and pathway analyses in 13,000+ species.
  • Knowledge-based visualizations: clustering, KEGG pathways, genome views, etc.
  • Download R code and data objects to reproduce the analysis,
  • Install locally as an R package or a Docker container (Windows, MacOS)

Old versions are still usable (v2.01, v0.96).
